# Cloud Task Manager

Cloud Task Manager is a simple full-stack task management application.

Users can add tasks, set due dates, mark tasks as completed, and delete tasks.  
The project was built to practice frontend development, backend API development, database usage, Git/GitHub, and cloud deployment.

## Features

- Add new tasks
- Set due dates
- Mark tasks as completed
- Delete tasks
- Save task data in a SQLite database
- Connect frontend to backend using Fetch API
- Deploy backend to a cloud platform

## Tech Stack

### Backend
